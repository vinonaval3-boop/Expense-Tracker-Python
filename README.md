# 📊 Expense Tracker (Python)

A clean and simple CLI-based expense management system built using Python, designed with CRM-like data handling and analytical capabilities. 
It allows you to record expenses, view summaries, generate analytics,  
and create charts such as **Bar Chart**, **Pie Chart**, and **Monthly Trend Line**.

---

## 🚀 Features

- ➕ Add new expenses  
- 📄 View all expenses  
- ✏️ Edit and delete saved expenses  
- 🔍 Search by date or category  
- 📅 Monthly expense summary  
- 📈 Analytics:
  - Highest expense  
  - Lowest expense  
  - Average spending  
- 🖼 Generates charts using Matplotlib:
  - Category-wise bar chart  
  - Category distribution pie chart  
  - Monthly trend line chart  
- 📂 Charts stored inside: `reports/charts/`
